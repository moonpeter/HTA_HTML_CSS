하이퍼링크 예제

HTML 페이지에서 다른 HTML 페이지를 연결하는 고리를 하이퍼 링크 라고 부릅니다. 하이퍼링크는 텍스트나 이지미에 다른 웹 페이지의 주소를 달아서 만듭니다. 하이퍼링크는 같은 사이트에서 다른 html 페이지뿐 아니라, 다른 사이트에 있는 웹페이지를 연결하기도 합니다. 하이퍼링크를 줄여서 링크로 부르기도 합니다.
#하이퍼링크 만들기
하이퍼링크(또는 링크)는 <a>태그의 href속성으로 만들며 이 속성을 이용해서 이동하고자 하는 목적지 문서의 주소를 지정합니다.
tile 속성은 말풍선 창에 나올 설명을 기입합니다.

  1. W3C 사무국(KOR) 방문
  2. https://w3schools.com/
  3. 네이버 연예
